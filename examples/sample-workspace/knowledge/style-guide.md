# Voice
| rule        | value                                  |
|-------------|----------------------------------------|
| person      | second person ("you")                  |
| tone        | plain, direct, no hype                 |
| sentence    | short; one idea per sentence           |
| forbidden   | "leverage", "seamless", "in today's…"  |

# Format
- Lead with the answer, then the detail.
- Prefer tables and lists over paragraphs.
- Max 1 example per point.

# Notes
This is reference material ("factory" in ICM terms): it is loaded by drafting
stages and stays stable across runs. It is `audience: both` because a human
reviews the voice rules. The table carries the truth; this prose is only context.
