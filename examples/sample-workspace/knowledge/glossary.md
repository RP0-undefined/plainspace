# Terms

| term      | definition                                                        |
|-----------|-------------------------------------------------------------------|
| concept   | One markdown file of durable knowledge. Has a `type`.             |
| stage     | One numbered folder = one pipeline step. Has a `_stage.md`.       |
| factory   | Reference material, stable across runs (`knowledge/`).            |
| product   | Per-run working artifacts (a stage's `output/`).                  |
| review gate | The point between stages where a human may edit `output/`.      |
